# CLI Reference

##

<br/>




* There is list of certain commands which are available in this plugin.
  Below is a little description of those commands.

<br/>

* <details>
  <summary>
    <strong> actions:add</strong>
  </summary>

  <br/>

  * This command will add actions to your **action instance**.

    * Pattern of command for adding action is something like this:

      <br/>

        ```shell
          $ node glue actions:add <Action-Folder-Name> <Name-of-Action> 
        ```

      <br />

    * For example if you wanted to create an action with folder named **myFirstAction** and action
        named **fileHandler**. You should enter the following command in the CLI:

        <br/>

        ```shell
          node glue actions:add myFirstAction fileHandler
        ```

        <br/>

  </details>
